什么是 Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。我们可以借助免费的 Gitpage 制作自己的博客网站。

案例网站：1.网站 A、2.网站 B、3.网站 C

需要的工具：Github 账户（点击查看注册教程）、魔法梯子（为稳定链接 Github）、Typora（编辑博文 Markdown 文件）、VPS（云服务器）

hexo 博客支持 Gitpage，即将博客上传至 Github 服务器，程序猿都有 Github，而且发布在 Github 又是免费的，相当多人选择 Gitpage。坏处是发布在 Github 的域名不能自选，且国内网络访问 Github 没有梯子的话登录比较不稳定，因此购买域名和 VPS 云服务器后发布在个人服务器上也是非常好的选择。当然，双修也是可以的。
